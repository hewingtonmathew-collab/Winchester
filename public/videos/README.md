# Hero Video Files

Place the following MP4 video files in this folder to enable full-screen video backgrounds on hero sections.
All videos should be dark/abstract tech themed, 1920×1080, loopable, with no audio track.

| File | Page | Recommended search |
|------|------|--------------------|
| `hero-home.mp4` | Home page | "dark particle network loop" |
| `hero-guardian.mp4` | Guardian OS page | "dark network nodes abstract" |
| `hero-services.mp4` | Services page | "digital data stream dark" |
| `hero-about.mp4` | About page | "dark technology abstract" |

## Free sources
- **Pexels** (pexels.com/videos) — search "dark abstract technology", download HD
- **Mixkit** (mixkit.co/free-stock-video/technology/) — free download, no attribution required
- **Coverr** (coverr.co) — free website background videos

Until video files are added, the animated particle canvas shows automatically as a fallback.
